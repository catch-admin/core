# CatchAdmin Frontend Utils

Use `web/src/support/` for shared frontend utilities.

- Use CURD composables for standard CRUD flows.
- Use `http` for custom requests and blob work.
- Use `helper`, `cache`, `message`, and `options` for shared frontend logic.

See [references/support-api.md](references/support-api.md) for the current module surface.
